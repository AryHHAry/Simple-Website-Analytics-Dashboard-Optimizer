# Simple Website Analytics Dashboard Optimizer

Versi Prototype / SaaS Prototype

"Simple Website Analytics Dashboard Optimizer" adalah alat analitik situs web yang sederhana, fokus pada privasi, dan tanpa cookie. Alat ini dirancang sebagai alternatif Google Analytics yang ringan dan esensial, khusus untuk blogger dan usaha kecil di Indonesia.

## Visi Proyek

Tool ini bertujuan membangun micro SaaS Simple Website Analytics Dashboard untuk Indonesia, membantu blogger/small business optimasi tracking privacy-focused, tanpa cookies, dengan analytics esensial dan scale ke vertical enterprise di 2026.

## Fitur Utama

- **Real-Time Tracking**: Script snippet untuk instalasi mudah.
- **Privacy-First**: Tanpa cookie atau penyimpanan IP (anonymized hash).
- **Geo/Source Insights**: Informasi negara dan sumber trafik tanpa melanggar privasi.
- **Custom Reports**: Filter tanggal dan ekspor data ke CSV/Excel/PDF.
- **Alert System**: Notifikasi cerdas untuk lonjakan atau penurunan trafik.
- **AI Anomaly Detection**: Prediksi tren masa depan menggunakan AI.
- **Content Optimizer**: Rekomendasi fokus konten menggunakan optimasi matematika (PuLP).

## Cara Menjalankan Secara Lokal

Pastikan Anda memiliki Python yang terinstal, lalu jalankan perintah berikut:

```bash
pip install -r requirements.txt
streamlit run app.py
```
